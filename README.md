# Antedimento Águas

# Instalação

1. Instale NodeJS direto do site oficial [node.js](https://nodejs.org/en/)
2. Instale o Expo após instalar NodeJS

   ```jsx
   npm install --global expo-cli
   ```
3. Instale as dependencias após clonar o projeto

   ```jsx
   npm install
   ```

6. Inicie o projeto

   ```jsx
   npx expo start
   ```
